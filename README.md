# OpenGL_BreakOutGame

## BreakOut Game
### 겨울방학 때, 수업 때 들은 그래픽스 수업을 복습 및 자세하게 알고 싶어서 만들었습니다. 부족한 실력으로 만들었습니다.
## 조작 방법
### 방향키 좌우 : 밑의 바를 좌우로 이동시킬 수 있습니다.
### 방향키 위 아래 : 볼의 속도를 높이거나, 방향을 앞 뒤로 바꿀 수 있습니다.
### r(소문자) : 재시작을 할 수 있습니다.
## 게임의 엉성한 부분
### 벽돌부시기 게임이지만, 공이 밑으로 내려가지는 않습니다.
